# Nova AI

An advanced, premium AI assistant powered by Google's Gemini API with multi-turn conversation, text-to-speech, web grounding, and persona-based responses.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set `GEMINI_API_KEY` in `.env` to your Gemini API key
3. Run the app:
   `npm run dev`

## Deploy on Vercel

1. Push to GitHub, import into Vercel
2. Add `GEMINI_API_KEY` as an environment variable
3. Deploy
